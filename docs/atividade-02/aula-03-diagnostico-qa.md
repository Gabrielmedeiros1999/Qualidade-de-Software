# 3. Práticas de QA Sugeridas

Sugira práticas que a startup pode adotar para melhorar a qualidade.

---

## 3.1 Lista de práticas

- Testes manuais das funcionalidades principais
- Registro e acompanhamento de bugs
- Revisão de funcionalidades antes da entrega

---

## 3.2 Explicação das práticas

Explique brevemente cada prática sugerida.

### Prática 1:
> Descrição:
Realizar testes manuais nas principais funcionalidades do sistema para garantir que estão funcionando corretamente antes de serem disponibilizadas aos usuários.

### Prática 2:
> Descrição:
Registrar todos os bugs encontrados em uma ferramenta ou sistema, permitindo o acompanhamento e correção adequada dos problemas.

### Prática 3:
> Descrição:
Revisar todas as funcionalidades para ver se estão de acordo com o que foi pedido.
